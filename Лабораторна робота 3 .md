- Command prompt - a program that provides user interaction with the operating system by entering commands in text mode.

- Shell - an environment that interprets commands entered by the user and passes them to the operating system for execution.

- Command - an instruction that performs a certain action in the command line environment.

**3.1 Яку базову інформацію надає рядок запрошення prompt?**
- The prompt line displays information about the current user, host name, current directory, and other data depending on the shell configuration.

**3.2 Для чого команді потрібні параметри та аргументи?**
Parameters and arguments allow you to configure the behaviour of a command, define input data and specific actions. Parameters usually start with <kbd>-</kbd> or <kbd>--</kbd> and arguments pass specific values to the command.

**3.3 Яке призначення команд ls, які параметри та аргументи вона може мати? Наведіть 3 приклади.**
Команда <kbd>ls</kbd>is used to display a list of files and directories in the current or specified directory.
- <kbd>ls -l</kbd> displays detailed information about files and directories.
- <kbd>ls -a</kbd> shows all files, including hidden ones.
- <kbd>ls /home/user/Documents</kbd> displays a list of files in the specified directory.
  
**3.4 Яким чином можна використати історію команд, які переваги це надає?**
- You can view the history of commands using the history command. This allows you to re-execute previous commands without having to re-enter them, which saves time and increases efficiency.  
**3.5 Яке призначення команди echo?** 
- <kbd>echo "Hello, World!"</kbd> prints the text "Hello, World!"
- <kbd>echo $HOME</kbd> displays the path to the user's home directory.
  
**3.6 Охарактеризуйте поняття змінної в оболонці Bash, які типи змінних вона підтримує?**
- Local - available only in the current session 
- Global - available in all child processes 
- Positional parameters - variables that contain script arguments 
- Special variables - system variables
  
**3.7 Яке призначення команд env, export та unset?**
- <kbd>env</kbd> – displays a list of environment variables.
- <kbd>export VAR=value</kbd> – makes the variable available to all child processes.
- <kbd>unset VAR</kbd> –  removes a variable from the environment.
  
**3.8 Які команди для отримання довідки по командам в терміналі ви знаєте?**
- <kbd>man <team></kbd> – opens detailed documentation
- <kbd>info <team></kbd> – provides extended information about the team.
- <kbd>--help</kbd> – a short help on how to use the command.
- <kbd>help</kbd> – built-in help for the Bash shell

## Контрольні запитання:

**1. Які типи команд існують в оболонці Bash?**  
There are four main types of commands in Bash:  
- **Built-in commands** – integrated into the shell (e.g., `cd`, `echo`, `exit`).  
- **External commands** – executable files located in the file system (e.g., `ls`, `grep`).  
- **Aliases** – user-defined shortcuts for commands (e.g., `alias ll='ls -la'`).  
- **Shell functions** – user-defined functions that execute a set of commands.  

**2. Що таке змінні оточення? Які вони бувають? Як їх можна переглянути в терміналі?**  
Environment variables are variables that affect the behavior of the shell and programs. They can be:  
- **System variables** (e.g., `PATH`, `HOME`, `SHELL`).  
- **User-defined variables** (custom variables set by the user).  
You can view them using the commands `env`, `printenv`, or `set`.  

**3. Опишіть змінну `$PS1`. Як в терміналі переглянути її вміст?**  
`PS1` is a variable that defines the appearance of the Bash command prompt.  
To check its value, use the command:  
```bash
echo $PS1
```  

**4. Як можна змінити значення змінної `$PS1`? Що при цьому відбудеться в рядку запрошенні в Bash? Як змінити значення цієї змінної не на поточний сеанс, а за замовчуванням?**  
You can temporarily change `PS1` like this:  
```bash
PS1="\u@\h:\w$ "
```  
This modifies the format of the command prompt. To make the change permanent, add the command to the `~/.bashrc` file.  

**5. Для чого використовують лапки в оболонці Bash?**  
- **Single quotes (`'`)** – disable interpretation of special characters.  
- **Double quotes (`"`)** – allow variable expansion.  
- **Backticks (`` ` ``) or `$()`** – execute a command and substitute its result.  

**6. Для чого використовують інструкції керування, які їх види Ви знаєте?**  
Control structures are used for conditional execution of commands:  
- **Conditional statements** (`if`, `case`).  
- **Loops** (`for`, `while`, `until`).  
- **Control operators** (`break`, `continue`, `exit`).  

**7. В чому різниця, якщо в кінці рядку запрошення Bash стоїть символ `$` чи `#`?**  
- **`$`** – indicates a regular user.  
- **`#`** – indicates a superuser (root).  

**8. Яке призначення команд `whereis` та `locate`? Яка між ними відмінність?**  
- **`whereis`** – locates the binary, source, and manual pages of a command.  
- **`locate`** – quickly finds files in the system using a pre-built index database.
